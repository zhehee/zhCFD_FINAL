- Python 3.7+
- matplotlib,numpy第三方库
- texlive/simple Tex

## 主要内容

- 关于期末大作业的报告
- 关于期末大作业的代码
- 关于期末大作业的代码生成的图片

- 在final.tex查看报告的 latex 源码
